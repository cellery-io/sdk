Cellery

Thank you for installing Cellery.

Run Cellery

Open a new terminal and run the following command to get start with Cellery:

  $ cellery [command]

Run the following command for Cellery help:

  $ cellery -h

Uninstall Cellery

Run the following command to uninstall Cellery.

  $ sudo bash /Library/Cellery/uninstall.sh
